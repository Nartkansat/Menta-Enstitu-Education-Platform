# 🌿 Menta Enstitu - Education & Management Platform

Menta Enstitu is a comprehensive platform developed to digitalize, accelerate, and provide a modern experience for educational and administrative processes from end to end. 

> **⚠️ Important Note:** This repository is created solely to showcase the project's presentation and architectural approach. Due to the commercial and private nature of the project, the source code is kept closed (Closed-Source).

**🌐 Live Project:** [mentaenstitu.com.tr](https://mentaenstitu.com.tr/)

---

## 🚀 About the Project

Menta Enstitu enables educational institutions to manage both administrative and academic processes from a single centralized point. It is developed adhering to modern web standards and best software engineering practices. 

**Clean Architecture** principles have been adopted to provide a scalable, secure, and high-performance infrastructure.

---

## 🏗️ Architecture & Folder Structure

The application follows a strictly decoupled Clean Architecture approach, ensuring maintainability and separation of concerns:

```text
📂 Menta Solution
 ┣ 📂 Menta.Api             # RESTful API Endpoints / Presentation
 ┣ 📂 Menta.Auth            # Authentication, Authorization & Identity Management
 ┣ 📂 Menta.Business        # Core Business Logic & Services
 ┣ 📂 Menta.Domain          # Core Entities, Enums & Domain Models
 ┣ 📂 Menta.Infrastructure  # Data Access (EF Core), DbContext & Repositories
 ┣ 📂 Menta.Shared          # Common Helpers, DTOs & Utilities
 ┗ 📂 Menta.Web             # UI / Client Application Integration
```

---

## 🏛️ Core Features

- **Advanced Admin Panel:** A centralized control point where FAQs, corporate information, logos, contracts, and other content can be dynamically managed.
- **Instructor Management:** Detailed control of instructor profiles, areas of expertise, and instructor-student matching systems.
- **Course & Group Management:** Dynamic course listing, curriculum management, and group-based custom access authorizations.
- **Modern & Layered Architecture:** A structure where Domain, Business, Infrastructure, API, and Web (UI) layers are designed independently and sustainably.
- **Rich & Interactive UI/UX:** User-friendly modern components offering seamless AJAX-based navigation, fully responsive design, and multi-language (localization) support.
- **Performance & Security Oriented:** Infrastructure compliant with SOLID principles, featuring centralized exception handling and JWT/Role-based security measures.

---

## 🛠️ Architectural Approach & Technologies

The project is built with a modern technology stack that meets the requirements of an enterprise-level application:

- **Backend:** High-performance API and business logic layers developed using C# and the .NET ecosystem.
- **Architecture Model:** Clean Architecture (business model supported by Domain-Driven Design approach).
- **Database Management:** Entity Framework Core (Code-First approach and optimized queries).
- **Frontend:** Modern CSS approaches (Tailwind CSS/PostCSS) and dynamic components offering a high-end user experience.
- **Design Patterns:** Dependency Injection, Repository Pattern, Request/Response (DTO) mapping, etc.
